## 📖 About The Project

O projeto consiste em um componente interativo de avaliação onde o usuário pode selecionar uma nota de 1 a 5 e visualizar uma tela de agradecimento após o envio.

Esse desafio foi excelente para praticar:

* Manipulação do DOM com JavaScript
* Eventos de clique
* Alteração dinâmica de conteúdo
* Controle de estados visuais
* Componentes interativos
* Estruturação responsiva

---

## ✨ Features

✅ Seleção interativa de notas
✅ Feedback visual ao selecionar opções
✅ Tela dinâmica de agradecimento
✅ Manipulação de DOM com JavaScript
✅ Layout moderno e responsivo
✅ Transições suaves

## 🧠 What I Learned

Durante esse projeto pratiquei conceitos importantes como:

* Manipulação do DOM com JavaScript
* Captura de eventos de clique
* Atualização dinâmica de conteúdo HTML
* Controle de exibição com `display`
* Trabalhar com `querySelector` e `querySelectorAll`
* Uso de variáveis CSS com `:root`
* Criação de componentes interativos e responsivos
